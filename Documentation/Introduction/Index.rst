Introduction
------------

This extension contains an indexer for ke_search which indexes sf_event_mgt records. Just install the extension
and configure the indexer for ke_search.

Updates and support
~~~~~~~~~~~~~~~~~~~

The source code of this extension is hosted on GitHub. Please report bugs and changerequests
directly at https://github.com/derhansen/sf_event_mgt_indexer

Updates will be published directly on TER.
